# WhatsApp Chatbot with Ollama and Twilio
This is a basic WhatsApp chatbot made with Ollama and Twilio in Python. It can remember history so that chats last.

The full tutorial for it's implementation and explanation can be found here - https://youtu.be/Sy0HtGY81mE

## Requirements
- Libraries - `pip install flask twilio ollama python-dotenv`
- A `.env` file is needed with the following tokens - `TWILIO_ACCOUNT_SID` and `TWILIO_AUTH_TOKEN`
